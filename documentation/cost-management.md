# Cost Management

This document describes how costs were controlled in this project.

## Cost Control Measures

- A **B1s** Linux VM was selected as a low-cost, burstable instance.
- Auto-shutdown was enabled to prevent unnecessary runtime.
- Azure Bastion was deployed only for testing and removed immediately afterward.
- Resources were deployed in a cost-effective region (Norwayeast).

## Result
The total cost of this project remained under $15.
